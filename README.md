# Calculator Application

A simple calculator application developed using JavaFX in Java.  
This project performs basic arithmetic operations through a graphical user interface.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Clear button
- Interactive GUI

## Technologies Used

- Java
- JavaFX
- NetBeans IDE

## Project Structure

```bash
calculatorapplication/
│
├── CalculatorApplication.java
└── README.md
```

## How to Run

1. Open the project in NetBeans or any Java IDE.
2. Configure JavaFX if needed.
3. Run `CalculatorApplication.java`.
4. Use the calculator GUI.

## Buttons Included

| Button | Function |
|--------|----------|
| 0-9 | Number Input |
| + | Addition |
| - | Subtraction |
| * | Multiplication |
| / | Division |
| = | Show Result |
| C | Clear Display |
